Aide

On peut créer des leçons, qui peuvent contenir des définitions, des dates ou encore des égalités.

Enregistrement

  • Pour enregistrer l'ensemble des leçons actuelles (ce qu'on appelle le pack de leçons), on peut les télécharger sous forme de fichier .txt via l'interface Import/Export.

    Pour accéder à un pack de leçons stockée sous forme de fichier .txt, il suffit de, dans l'interface Import/Export, sélectionner le fichier, puis de cliquer sur importer.

  • Une autre solution d'enregistrement consiste à créer un compte dans la page Compte, y créer une ou plusieurs sessions, et y enregistrer le pack de leçons actuel. Il suffira ensuite de se connecter pour avoir accès à toutes ses sessions.

    Pour accéder à une session stockée dans le compte, il faut se rendre sur la page Compte, se connecter si on ne l'est pas déjà, sélectionner la session et cliquer sur importer.

Attention : importer une session effacera le pack de leçons actuellement chargé.

Enfin, on peut se faire interroger : attention, le mode Restitution Écrite nécessite de taper la définition correctement, sans reformuler ou changer les mots.
