Pack de leçons

Paramètres

Couleur:

Aide

En résumé :
  1. Tu bosses sur l'Accueil.
  2. Tu stockes dans ton Compte.
  3. Tu récupères tes archives quand tu changes de matière.

PAGE : L'ACCUEIL

C'est ton "établi" : là où tu crées, tu modifies et tu révises tes leçons en direct.
  • Le "Pack de Leçons" : C'est ce que tu as "sous la main". Tes données y restent même si tu fermes l'onglet.
  • Onglet "Créer une leçon" : Tu donnes un titre (ex: 1ère Guerre Mondiale) pour organiser tes futures fiches.
  • Onglet "Ajouter" : Tu remplis ta leçon avec 3 types de contenus :
    • Définitions (Concept : Explication)
    • Dates (Année : Évènemment)
    • Égalités (Formule A : Formule B)
  • Onglet "S'interroger" : Choisis une leçon et lance un test.
    • 4 Modes : QCM, Vrai/Faux, Écrit...
    • Mode Conseillé : L'Auto-validation. Tu réponds, tu vérifies, et tu décides si tu as le point. Le top pour apprendre vite.

PAGE : LE COMPTE

C'est ton "archiveur" : là où tu sauvegardes tes cours sur le cloud pour ne jamais les perdre.
  • La "session" : C'est un dossier permanent (ex: Session SVT ou Session Maths).
  • Créer une session : Prépare un grand tiroir vide dans ton compte.
  • Ajouter le pack actuel : Envoie tout ce qui est sur ta page d'accueil dans une session choisie.
    • Note : Si une leçon existe déjà dans la session, elle est mise à jour avec tes nouveaux ajouts.
  • Importer une session : Récupère une session enregistrée et remplace ton pack actuel par elle.
    • Attention : Cela écrase ton travail en cours sur l'accueil, donc si tu veux le garder, enregistre le avant !
